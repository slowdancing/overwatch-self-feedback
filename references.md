# 레퍼런스 / 지향하는 플레이스타일

## 맵 인지 / "탑뷰" 감각 만들기
오버워치엔 적을 보여주는 미니맵이 없음 → 탑뷰는 화면이 아니라 **머릿속에서 재구성**하는 능력.

**4가지 입력**
1. **소리**: 발소리·스킬음·리로드음의 방향/거리. 둠 어그로 시 "뒤에서 몇 명 오는가" 판단의 핵심.
2. **킬로그 + Tab**: 킬로그 = 그 자리에 생긴 빈 공간(침투각). Tab = 생존자/궁 상황, 소거법으로 힐러 위치 추정.
3. **마지막 목격 위치 + 경과 시간**: "몇 초 지났으니 지금쯤 여기" 추정 → 맵 동선 암기가 토대.
4. **스킬/궁 소모 추적**: 상대가 방금 쓴 쿨기 = 다음 몇 초간 못 하는 것 = 내 진입 안전 여부.

**훈련법**
- 리플레이를 **적팀 시점 + 탑뷰(자유 카메라)** 로 다시 보며 "내가 몰랐던 적 동선" 확인. (탑뷰 감각 직접 훈련)
- 한타 중 1초에 한 번 Tab 힐끗.
- 진입 전 자문: "나를 보러 올 수 있는 적 몇 명? 이탈 경로 어디?"
