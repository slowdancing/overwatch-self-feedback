# 레퍼런스 / 지향하는 플레이스타일

## 탱커 기본기 (둠피스트 외 메인탱 — 마우가·시그마 등)

**스페이스 만드는 방식이 둠피와 다르다**
- 둠피 = **다이브 탱**: 밀어내고 → 빠지고(리셋). 인앤아웃이 정상.
- 마우가·시그마 = **앵커/브루저 탱**: 버티는 존재감으로 스페이스 유지. 빠지면 자리 내줌.
- "적이 쉽게 들어온다" 느낌 = 앵커 탱을 둠피처럼(빠졌다 들어갔다) 플레이해서 리셋할 때 자리 비는 것. → **눌러앉아 위협을 줘야 함.**
- 원칙: **스페이스는 '위치'가 아니라 '위협'으로 지킨다.** 들어오면 응징한다는 위협이 있어야 안 들어옴.
- 단, 무조건 사수 X. 지킬 가치 있는 자리(고지대·초크·목표)만. 못 버틸 자리 붙잡으면 던지는 것. **시그마는 특히 고지대 사수** (킷 전체가 높이에서 나옴).

**⭐ 자리 먹기 운영 모델 (최종 목표)**
> **자리를 먹고, 최대한 앞에서, 유지하면서 압박한다. 단 많이 맞지 않는다.**
- **"있는 것만으로 압박이 된다"** — 자리를 먹은 상태 자체가 밸류. 킬도 딜도 필요 없음. 내가 거기 있으면 **상대는 그 공간을 못 씀.**
- **⭐⭐ '자리'의 정의: 내가 상대를 때릴 수 있는 곳이 아니라, 상대가 나를 때릴 수 없는 곳.**
  - "때릴 수 있는 자리"를 찾으면 → 상대 앞에 서게 됨 → **상대도 나를 때림** → 맞으며 딜 넣는 소모전 → 힐 자원 태우고 밀림.
  - "안 맞는 자리"를 찾으면 → **거기 있는 것만으로 상대가 그 공간을 못 씀** → 압박 유지, 비용 0. (어차피 딜은 부산물)
  - **실전 질문 교체**: ❌ "어디서 때릴 수 있지?" → ✅ **"어디에 있으면 상대가 날 못 때리지? 그리고 그게 상대한테 불편한가?"** (두 조건 다 필요 — 안 맞는데 상대가 신경도 안 쓰면 그냥 숨은 것)
  - → 노트 전체가 사실 이 정의였음: **벽에서 벽으로**(안 맞는 경로) / **CC엔 정면 금지**(때릴 각 안 주기) / **양각 회피**(때릴 수 있는 방향을 1개로) / **많이 맞지 말 것**.
- **상대 반응별 분기 (뭐가 와도 내가 이김)**

| 상대 반응 | 내 대응 | 결과 |
|---|---|---|
| 아무도 안 옴 | 그대로 유지 | 자리 계속 먹음 (공짜) |
| 힐러·딜러 1~2명 옴 | **걷어낸다** | 그 자리 뺏음 |
| 상대 전부가 내 쪽으로 옴 | **도주** | 적 전체를 끌어낸 것 = 다른 쪽이 5:0 |

- **"많이 맞지 말 것"의 이유**: 맞아 죽으면 자리도 같이 잃음. **안전 = 자리 유지 시간 = 밸류** (안전이 곧 일하는 방법).
- **⚠️ "최대한 앞"의 정의 = 도주로가 살아있는 가장 앞.** 도주로가 끊기면 그건 '앞'이 아니라 '고립'. (3번 분기[전부 오면 도주]가 성립하려면 도주 가능해야 하므로, **도주로 확보가 이 모델 전체의 전제 조건**)

**답답하면 사이드로 돌아 '각을 벌린다'** — 검증됨(눔바니)
- 정면에서 강펀 변수를 쥐어짜는 건 **각 1개로 뚫으려는 것.** 사이드로 돌면 **각이 2~3개**가 되고 상대는 세 방향을 동시에 못 막음.
- **내가 딜을 더 넣는 게 아니라, 팀 전체의 딜이 통하게 만드는 것.** (탱커가 이기는 방식)
- 단 혼자 안 됨 — **아군이 내 각을 따라오지 말고 각을 벌려줘야** 완성.
- 판단 신호: **"구도가 답답하다"** 싶으면 억지로 밀지 말고 즉시 사이드.

**죽을 거면 빨리 죽어야 리그룹이 된다**
- 산발적으로 물려 죽으면(트리클) 리스폰이 어긋나서 **영원히 5:5가 안 됨.** 차라리 같이 죽으면 같이 나와 온전한 한타 가능.
- 지는 한타에서 질질 끄는 게 오히려 손해. (입구막 풀 때도 같은 원리)

**⚠️ 양각 잡을 때 — 상대 탱이 나한테 붙으면 조심 (궁 없으면 도주)** — 검증됨(하바나 3거점)
- 양각이 통하는 이유: 내가 상대 시선 한쪽을 잡아 못 들어오게 함. **BUT 상대 탱이 나한테 붙으면 구도가 뒤집힘.**
- 양각 자리는 본대에서 떨어진 옆·뒤 → 상대 탱이 오면 **아군 지원에서 먼 곳에서 1대1 = 고립.** 탱 대 탱은 안 끝나니 그동안 물려 죽음.
- **궁 있으면 OK**(고립을 리셋하는 카드) / **궁 없으면 위험.**
- 궁 없을 때 답: **버티지 말고 각 포기하고 빠진다.** 그 양각 자리는 "탱이 안 올 때만 유효." (= 자리 먹기 모델 3번 분기의 축소판: "탱이 오면 궁 없을 때 도주")
- 예방: 양각 자리 잡을 때 **탱 붙으면 빠질 도주로**를 미리 확보 → 궁 없어도 안전. ("최대한 앞 = 도주로 살아있는 가장 앞")

**⚠️ 사이드 어그로에 '끌리지' 않기** — 어그로를 *끄는 것*과 어그로에 *끌리는 것*은 정반대
- **내가 어그로를 끈다** = 내가 자리를 정하고 상대를 부름 → **주도권 나에게.**
- **내가 어그로에 끌린다** = 상대가 미끼 던지고 내가 따라감 → **주도권 상대에게.** 사이드에서 깔짝대는 것에 반사적으로 반응하면 **먹어둔 자리를 스스로 버리는 것.** 상대는 딜러 하나로 탱커를 끌어낸 이득.
- 자리 먹기 모델의 정반대: 모델은 *"내가 자리 먹으면 상대가 온다"*(상대가 움직임) / 끌림은 *"사이드에서 부르니 내가 간다"*(내가 움직임).
- ※ 아래 "혼자 도는 애 걷는 건 거래"보다 더 나쁨 — 그건 *내가 판단해서* 간 것, 이건 *반응해서 끌려간* 것.

**자리를 밀 때 vs 지킬 때**
- 상위 원칙: **이미 잡은 좋은 자리는 그 자체로 가치.** 미는 건 *현재 자리를 새 자리와 바꾸는 것* → **새 자리가 더 좋을 때만 민다.**
- **혼자 사이드 도는 애를 걷는 건 '이득'이 아니라 '거래'**: 내 자리(탱 존재감)를 내주고 걔 하나 잠깐 밀어내는 거래. **대개 손해** (본대의 탱 1명 > 사이드의 딜 1명). 게다가 새로 얻는 자리가 아예 없음 = 그냥 자리 버리기.
  - "몰아내고 복귀해서 도와준다"가 성립하려면 복귀 때 자리가 남아 있어야 하는데, 그 사이 이미 밀려 있음.
- **판단 질문: "내가 뜨면 뭘 잃나?"**
  - ✅ 가도 됨: 한타 사이 **다운타임**(비용 0) / 걔가 우리 뒷라인에서 **공짜 밸류** 뽑고 아무도 못 막음 / **확실히 빨리** 잡을 수 있음(기동기 X, 고립·저피)
  - ❌ 가면 안 됨: **본대 자리싸움 진행 중**(내 부재가 제일 비쌈) / 상대가 **미끄러운 애**(트레·겐·솜, 못 잡고 시간만) / 내가 **핵심 지형(초크·고지대) 점유 중**(뜨면 헌납) / 우리 뒷라인이 **알아서 버팀**

**양각(크로스파이어)에서 맞을 때 — 정적 흡수 금지**
- ❌ 자리 유지하며 힐만 받기 = 최악 (적 둘 딜 vs 힐러 자원, 나는 무행동. 갇힌 정적 탱 = 즉사각).
- ✅ ① **한 각을 민다** (팀 팔로우업 되는 각). 그쪽이 나 상대하느라 크로스파이어 풀림.
- ✅ ② **엄폐로 한 각 사이트라인 끊기** → 1각 싸움으로 전환 (사이트라인 관리의 탱커 버전).
- 못 이길 양각이면 '버티며 힐' 말고 **엄폐로 각 끊고 리셋.** 항상 능동적 각 정리.

**솜브라 대응 (탱커 버전)** — [[log]] 네온 교차로 참고
- 해킹당해도 안 죽는 위치 / 해킹 쿨(~8s) 창에 스킬 / 팀 근처 유지(솜브라 응징 유도) / 안 되면 안 끊기는 탱(오리사·라인·자리야) 스왑.

**다이브 조합(윈스턴+트레 등) 상대 — 둠피는 답이 아님** — [[log]] 할리우드 참고
- 윈스턴은 기동성 좋아 **안 걷어짐**(둠피로 압박 헛수고). 승부처는 상대 탱이 아니라 **내 뒷라인 피(peel)**.
- 둠피 = 다이브 탱이라 피가 안 됨 → **자리야로 스왑**(버블로 피 + 차징 → 윈스턴 순삭). 대안 오리사·시그마.
- 원칙: **못 걷는 탱(호그=무거움 / 윈스턴=기동성)은 압박 대상이 아니다.** 시선을 딴 데로.
- **⭐ 마인드 스위치 — 사냥꾼 → 경호원**: 상대가 풀 다이브면 **"우리팀 근처에서 지킨다"** 는 마인드로 플레이. 둠피처럼 앞·옆으로 파고들지 말고, 다이브 올 각과 뒷라인 사이에 붙어서 → 다이브 오면 돌아서 응징 → 다이브 밸류를 0으로. (둠피=파고드는 사냥꾼 / 안티다이브 탱=붙어서 지키는 경호원, 시선 방향이 반대)
- 안티다이브 우선순위: **자리야(아군 직접 버블로 구함) > 오리사·시그마.** 완벽한 안티다이브는 피 서폿(브리·키리코)까지 있으면 쉬워짐(단 내 통제 밖 → 내 레버는 탱 스왑 + 경호원 포지션).

**다이브 상대, 세 가지 선택지**
1. **안티다이브(자리야) — 정석·높은 확률.** 가위바위보상 안티다이브 > 다이브. 구조적으로 유리한 싸움.
2. **맞다이브(둠피+트레) — 유효한 Plan B, 코인플립.** 둠피+트레의 순간 폭발력 > 윈+트레는 맞음. 하지만: (a) 윈스턴은 공격+피 둘 다 되는데 둠피는 공격만(수비 절반 불리), (b) 둠피 다이브는 고분산, 윈스턴은 꾸준, (c) 결국 "우리 트레 vs 상대 트레" 개인기 싸움.
3. **기준**: 맞다이브는 **내 트레+콤보 정확도 확신 + 나머지 뒷라인도 다이브 버팀**일 때만. 확신 없으면 자리야가 기댓값 높음. (유리한 매치업 버리고 코인플립 고르는 것 = 실력으로 이겨야 하는 싸움)

**둠피 픽 판단 (상대에 플랭커 있을 때)** — 기준은 "상대 서브딜러 유무"가 아님
- 진짜 질문: **"내가 둠피로 빠지면 우리 뒷라인이 버티나?"**
  - **버팀** (뒷라인 기동기 O / 피 서폿[브리·키리코] O / 상대 플랭커가 뒷라인 위협 못 함) → 둠피 OK, out-tempo로 상대 서폿 먼저 자르는 이득도.
  - **못 버팀** (위도우·솔저처럼 기동기 X + 피 서폿 X) + 상대 다이브 → 둠피는 짐 → **자리야.**
- **상대 탱 종류가 더 중요**: 상대 탱이 윈스턴/볼(다이브)이면 안티다이브 필요. 라인/오리사/호그(홀드)면 둠피 유효.
- **습관 교정**: "둠피로 먼저 상대 서브딜러 걷으러 감"은 트레·겐·솜 상대면 **벤처 추격의 탱 버전**(시간 낭비+본대 이탈). 둠피 기동성 = 사냥용 X, 스페이스 창출/리셋용. (단 뒷라인 *근처*에서 다이브 온 트레 역행 빼고 잡기는 OK)

## 오리사 노트
둠피보다 체급이 세서 상대적으로 편함. 궁 외 쿨기가 즉발·CC 무시라 둠피의 "패닉/끊김" 문제가 적음 (솜브라·CC 상대 안정적).

**생존 쿨기 로테이션 (핵심)**
- **E(재블린 스핀, 투사체 씹기) 먼저 → Shift(포티파이: 뎀감 + CC무시) → Shift 끝날 때쯤 쿨 돈 E 다시.**
- 이렇게 **번갈아 겹쳐** 쓰면 방어 지속시간이 길어져 오래 버팀.

**공격적 활용**
- 오리사는 체급 강력 + E로 투사체를 씹으므로, **상대 스킬이 빠졌으면 과감하게 E로 진입해 자리를 밀어내는** 플레이도 좋음.
- (= 탱커 기본기의 "위협으로 스페이스 지키기" + "상대 카운터 쿨기 빠진 창에 투자" 원칙과 동일.)

**자리 미는 방식의 정체성 (둠피 vs 오리사)**
- **둠피 = 폭발적 변수**: 펀치 밀치기 + 지진강타로 자리 먹고, 블락으로 강펀 채워 **밀치기 + 데미지 변수력**으로 자리를 밀어냄. (버스트·다이브)
- **오리사 = 지속 탱싸움**: 천천히 탱 대 탱을 하며, 상대 스킬 빠지면 **E+Shift+E 로 체급으로 밀어냄.** (소모전·앵커)
- → 같은 "자리 밀어내기"라도 **둠피는 한 방, 오리사는 눌러앉아 조금씩.**

**대(對) 다이브 / 사이드**
- 사이드 파는 적·다이브 탱커가 있으면: **나머지 적들을 홀딩(앵커)** 하면서 **투창(에너지 재블린)을 한 번씩 던져** 견제·응징.
- 즉 오리사는 앵커로 정면을 잡아주는 게 본업 — 다이브를 쫓지 말고 홀딩 + 투창 피킹.

**적이 나를 패싱/스플릿할 때 (오리사 수비)**
- 적 일부가 2층 등으로 패싱 = 적이 팀을 쪼갬. **탱커가 그쪽으로 갔으면 앞에 남은 적(서폿)은 무방비.**
- 오리사는 **위로 쫓지 말 것**(느림, 자리 버림). 대신 **탱 없는 앞라인을 묶기 → 아예 잘라내기**(서폿 자르면 플랭커도 힐 잃음). 쪼개진 적의 **탱 없는 쪽 = 약점.**
- 조건 2개: (1) **2층 플랭커는 우리 팀(히트스캔·기동)이 맡는지 확인** — 아무도 안 맡으면 피(투창/스핀)로 전환. (2) **2층에 등 내주지 말 것** — 엄폐·포티파이.
- 한 줄: **패싱당하면 = 탱 없는 쪽이 약점, 거길 친다. (플랭커는 팀이 맡고, 등 안 내주는 조건)**

**🎯 훈련: 투창(에너지 재블린) 명중률** — 오리사 실력의 상한
- 투사체 = **이동 타겟 리드샷**(예측 조준) 필요. (겐지 표창과 같은 감각)
- **벽에 꽂으면 추가 데미지 + 스턴** → 벽 낀 상황에서 밸류 폭증 (내 벽 끼는 습관과 시너지).
- 워밍업/데스매치에서 의식적으로 투창 조준 연습. 용도: 다이버 응징 / 마무리 / 원거리 견제.

## 둠피스트 테크닉 노트

**강펀치를 '맞히는 기술'로만 보지 않기** (네팔 질풍 교훈과 같은 결)
- **헛펀치 → 상대 진영으로 빠지기**: 강펀치가 빗나가면 무리해서 복귀하지 말고, 아예 상대 진영 쪽으로 빠져 침투/이탈각을 잡는 것도 좋은 플레이. (잘하는 둠피 선수들도 이렇게 함)
- **페이크 강펀치로 자리 걷기**: 자리를 밀어낼 때 꼭 강펀치를 *맞혀서* 걷을 필요 없음. **쓰는 '척'** 만 해도 상대가 물러나 자리가 밀림.
- **맞둠피 상황**: 상대 둠피는 내 펀치를 스킬로 피하려 함 → 페이크 펀치로 **상대 스킬을 먼저 빼낸 뒤** 진짜 각을 잡기.
  - 기본: **상대 둠피부터 걷어(압박)** 탱 듀얼 우위 잡기. **단 거리(대형)가 깨진 혼전이면 → 뒷라인으로 침투.** (구조 무너지면 뒷라인 열림)

**파워블락 위치 = 강펀의 용도를 정한다**
- **개활지에서 블락 쓰지 말 것.** 개활지에서 쓰면 강펀이 차도 위험해서 **바로 도주용으로 태워야** 함 → 변수 못 냄.
- **벽/엄폐 낀 위치에서 블락** → 블락으로 찬 강펀을 도주에 낭비 안 하고 **변수용으로 아낌.**
- **⭐ "벽에서 벽으로" — 최대한 벽에 붙어서 블락, 동선 전체를 벽 단위로 설계.** 개활지를 지나가지 말고 벽↔벽으로 뛴다. (블락 위치만의 문제가 아니라 **개활지에 머무는 시간을 0으로** 만드는 것)
- (네온 교차로 "솜브라 있으면 블락 벽 끼고"의 일반화 — 솜브라 없어도 개활지 블락은 강펀을 도주에 낭비시킴.)

**파워블락 타이밍** — 블락은 "막은 순간"이 아니라 "실제로 흡수한 데미지"로 강펀이 참(정면 ~100 피해 흡수 → 강화)
- 상대가 **쳐다볼 때 X, 실제로 쏘기 시작할 때 O.** 눈맞춤에 반사 블락하면 흡수할 딜 없어 강펀 안 참 + 시간 낭비.
- **히트스캔**(솔저·캐서디·위도우·바스티온): 즉시 도달 → **반 박자 예측해서 먼저** 켜기.
- **투사체**: 이동 시간 있음 → **날아오는 거 보고** 켜도 됨.
- **많이 맞을 때 켜기**: 여러 명 화력 쏠릴 때 / 고딜 소스(호그 근접·바스티온) → 강펀 순식간에 참. 약한 견제 한 발엔 낭비.
- ⚠️ **역이용 주의**: 고수는 블락 보면 사격 멈추고 기다림 → 블락 시간만 날림. **딜이 불가피하게 들어올 때만** 켜기.
- 한 줄: *벽 끼고 자리 먹은 뒤 → 화력이 실제로 나한테 쏠리는 순간 블락. 히트스캔이면 반 박자 먼저, 투사체면 보고.*

**강펀 채울 때 적이 안 보이는 문제 — 정보와 충전을 동시에 하려니 안 됨**
- 원인: 안전하려고 **벽을 끼면 벽이 시야도 막음.** → 강펀 다 차고 고개 드는 순간 상황이 바뀌어 타겟을 모름.
- **고침 1: 순서 바꾸기.** ❌ 벽 끼고 충전 → 다 차고 "누구 치지?" / ✅ **먼저 적 위치 파악 → 타겟 정하고 → 그 각 잡고 충전 → 차자마자 꽂기.** (= 왕의 길 나노 교훈과 동일: 받고 나서 찾으면 늦다)
- **고침 2: 블락 자리는 '완전 엄폐'가 아니라 '모서리'.** 몸은 가려지되 한쪽 시야는 살아있는 자리(기둥·모서리 피킹). 안전 + 정보 동시 확보.
- **고침 3: 고지대 (최선).** 시야 + 펀치각 한 번에 해결. 퍼진 적이 위에서는 다 보이고, **낙하 강펀치** 각도 나옴.
- **"적이 퍼져있다"는 사실 기회**: 퍼짐 = 서로 지원 안 됨 = **각각 고립.** 둠피에겐 좋은 조건. 문제는 "칠 사람이 없는 것"이 아니라 **"닿을 길이 안 보이는 것"** → 고지대·측면 루트로 **닿는 길** 확보. 그래도 각 없으면 **자리 밀기로 전환.**

**동선 발소리 관리 (특히 호그 상대)**
- 호그는 발소리 듣고 그랩으로 카운터함. 네팔 오른쪽 방처럼 **소리 큰 길**은 위험.
- 대응: **앉아서 발소리 줄이기**, 또는 **소리 흘리고 잠깐 쉰 뒤(페이크) 진입** → 그랩 타이밍 흐트러뜨림.
- (= "정보"의 반대편 — 내가 상대에게 주는 정보를 관리하는 것.)

**천천히 못 걷는 지형의 예외** (도라도 "탱커 걷어 자리 확보" 원칙의 예외)
- 서킷 로얄, 도라도 초반 공격처럼 **탱커를 천천히 걷을 수 없는 초크/지형**이 있음.
- 이럴 땐 **아예 뒤로 돌아 트레이서처럼 어그로를 끌어 입구(초크) 압박을 풀어주기.**
- 단 진입 전 **팀 서브딜러 위치 체크** 필수.

**콤보 — 지진강타 낙하 강펀치**: 강펀치 차 있을 때 **지진강타를 일부러 2층(고지대)에 찍고 → 낙하하며 1층 적에게 강펀치 꽂기.**
- 이점 1: **위에서 오는 각** = 적이 예측 못 하는 시선 밖 기습 ("상대가 안 볼 때 꽂기"의 지형 버전).
- 이점 2: 지진강타로 **없던 높이를 스스로 창출** → 2층 없는 상황도 만들어 씀.

**⭐ CC 있는 상대에겐 정면으로 지진강타 쓰지 않는다**
- 대상: **아나(수면총), 캐서디(섬광탄), 브리기테(방패 강타), 시그마(바위), 호그(그랩), 키리코(스즈)** 등.
- 이유: 지진강타는 **궤적이 뻔하고 착지 지점이 예측됨** → 정면이면 조준하고 기다리기만 하면 됨. 정면 진입 = 상대에게 **직선 사거리 + 충분한 반응 시간**을 다 주는 것.
- **대신 사이드로 돌아서 지진강타** → 각이 꺾여 CC 명중이 훨씬 어려워짐.
- (= "카운터 쿨기 살아있으면 돌진 금지"의 실행 동작. 빼거나 못 빼면 최소한 **각을 꺾어서** 들어간다.)

**강펀치는 거리를 두고 날린다 (근접 X)**
- 가까이서 각 잡으려 하면 **CC 사거리 안에 먼저 들어감** → 섬광탄·방패강타·바위 맞고 끝. CC는 대부분 근거리 스킬.
- 강펀치는 **원거리 진입기** — 이미 붙어 있으면 그 거리가 무의미해지고 차징 노출만 됨.
- 거리가 있으면 상대 **반응 시간을 뺏음**. 가까이서 차징하면 상대 시야에서 다 보여 섬광 준비하고 기다림. (= 도라도 "길게 차징하면 시온이 확인하고 피함"과 같은 얘기)

**⚠️ 낙사 구역 근처에선 강펀치 금지**: 타겟 판단이 아무리 좋아도 낙사하면 전부 무효. 각 보기 전에 **내 발밑(낙사각)부터 확인.** (오아시스 패인)

**걷어도 의미 없는 대상 — 벽타기 영웅**: 한조·키리코처럼 **벽타기 있는 애들은 2층에서 걷어내도 다시 올라감** → 걷는 행동 자체가 무의미. (= "안 걷어지는 대상은 압박 대상이 아니다"의 또 다른 형태)
- **그럼 뭘 하나 → 아예 뒤로 돌아 양각을 잡아준다.** 밀어내는 게 안 되면 **각으로 자리의 가치를 떨어뜨린다.** (2층이 강한 건 한 방향에서만 상대해서 안전하기 때문 → 뒤/옆 각이 생기면 안전하지 않게 됨)
- **⭐ 돌았을 때는 "변수" 욕심 버리고 "어그로"만 끈다 — 강펀이 차 있어도.**
  - 뒤로 돈 순간 **밸류는 이미 발생 중**: 상대 시선이 나한테 옴 → 본대 압박 풀림 + 아군 딜각 열림. **아무것도 안 죽여도 일하고 있는 것.**
  - 여기서 "강펀 찼으니 뭐라도 잡자"고 욕심내면 → 혼자 물려 죽고 **어그로도 사라지고 본대는 탱커까지 잃음.** 얻은 걸 전부 반납.
  - **적용 맵**: 블리자드 월드 1거점, 도라도, 서킷 로얄 등 **초크가 빡센 곳** (= "천천히 못 걷는 지형" 대응의 확장판)

**기동성 영웅 상대 펀치 난이도**: 겐지·디바 등 기동성 좋은 영웅이 나오면 강펀치 명중이 어려움 → 펀치 변수에 의존하지 말고 자리·양각 쪽으로 가치 전환.

**매치업 주의 — 퀸 칼**: 퀸 칼(상흔) 맞은 상태로 지진강타 이탈 시 도로 끌려올 수 있음. 주의.

## SUEL vs PROGEN — 상충이 아니라 상황별 두 도구
- **SUEL = 뒷라인 괴롭히기 (고위험 고보상)**: 뒷라인이 접근 가능/고립 + 셋업(쿨기·이탈로·시선분산) 갖춰졌을 때. 픽 하나가 결정적일 때.
- **PROGEN = 정면·탱커 압박 (안전한 기본값)**: 뒷라인이 보호돼 침투 시 고립될 때, 상대에 플랭크 응징 수단(호그 그랩·아나 수면·히트스캔) 있을 때. "탱커 친다"의 가치는 탱딜이 아니라 **어그로/스페이스 + 디플렉트로 쿨기 빼기 + 크로스파이어 유지.**
- **라벨은 '디폴트 성향'이지 배타적 X**: PROGEN도 리그룹 타이밍엔 사이드 감(=SUEL식), SUEL도 셋업 됐을 때만 뒷라인 감. → **통합: "기본은 PROGEN, 각 나오면 SUEL."**
- **⭐ 나에게 맞는 순서**: 내 죽음 원인은 전부 "셋업 없이 너무 깊이 들어감"(네팔 개활지 고립). 그래서
  - **지금 = PROGEN 토대 먼저** (정면·연결·얕은 양각·쿨기 빼기 → 고립사 교정).
  - **나중 = SUEL 모드 해금** (뒷라인 괴롭히기는 '모두의 위치'를 알아야 안전 → 내 훈련 중인 위치 파악이 완성되면 열림).

## 후퇴 페이크 (빼는 척 → 들어가기) — 둠피·겐지 공통
빼는 순간 상대의 **경계·시선·쿨기가 한 번에 풀린다** (위협 사라졌다고 판단 → 방심하고 쫓아옴, 시선은 도망가는 나에 고정, 방어 쿨기 안 켬). 그 무방비 순간을 친다.
- **둠피**: 빼는 척 → 파워블락으로 강펀 채우고 → 돌아서 강펀치로 변수. 쫓아온 상대는 무방비.
- **겐지**: 빼는 척 → 순간 벽 타서 → 쫓아온 상대 **위에서** 암살. (시선은 아래, 나는 위)
- **특히 "지는 한타/교환 노리는 상황"에서 값짐**: 그냥 죽으면 0. 쫓아온 1명이라도 끊으면 → 동수 교환 or 다음 한타 인원 우위. **죽더라도 밸류 뽑고 죽기.**
  - ※ 왕의 길의 "동수 교환은 손해"와 모순 아님: 그건 *내가 자원 쓰고 이득 없이* 교환할 때. 이건 *지고 있을 때 상대를 끌어들여* 만드는 교환. 상황 뒤집히면 원칙도 뒤집힘.
- **경계**: 페이크에 취해 진짜 빠져야 할 때 안 빠지지 말 것. 상대가 **실제로 방심하고 쫓아왔을 때만** 돌아선다. 안 쫓아오면 그냥 빠지기.

## 🗺️ 맵별 요충지 — "선을 그을 위치"
"선을 긋는다"는 방법은 익혔으니, 이제 **맵마다 어디에 그을지**를 모은다. 맵 걸리면 여기부터 확인.

| 맵 | 요충지 / 운영 |
|---|---|
| **루나사피** | **후반부 방 + 1.5층** — 여기서 압박해야 상대 딜각이 죽음. 진입은 우클→점프 가속→지진강타 (**커스텀에서 연습 필요**, 급하면 가속 안 실림) |
| **네팔** | 거점 입구에 선 긋고 벽 끼기. **오른쪽 방은 발소리 큼** → 호그 있으면 앉아서 or 페이크 |
| **블리자드 월드 1거점 / 도라도 초반 / 서킷 로얄** | 초크가 빡세 **천천히 걷기 불가** → 뒤로 돌아 어그로로 입구 압박 풀기 (어그로 중엔 변수 욕심 X) |
| **왕의 길 3거점** | 개활지라 사각 강펀 구간이 적음 → 펀치 변수 기대 낮추고 자리·양각으로 |

## 맵 인지 / "탑뷰" 감각 만들기
오버워치엔 적을 보여주는 미니맵이 없음 → 탑뷰는 화면이 아니라 **머릿속에서 재구성**하는 능력.

**4가지 입력**
1. **소리**: 발소리·스킬음·리로드음의 방향/거리. 둠 어그로 시 "뒤에서 몇 명 오는가" 판단의 핵심.
2. **킬로그 + Tab**: 킬로그 = 그 자리에 생긴 빈 공간(침투각). Tab = 생존자/궁 상황, 소거법으로 힐러 위치 추정.
3. **마지막 목격 위치 + 경과 시간**: "몇 초 지났으니 지금쯤 여기" 추정 → 맵 동선 암기가 토대.
4. **스킬/궁 소모 추적**: 상대가 방금 쓴 쿨기 = 다음 몇 초간 못 하는 것 = 내 진입 안전 여부.

**훈련법**
- 리플레이를 **적팀 시점 + 탑뷰(자유 카메라)** 로 다시 보며 "내가 몰랐던 적 동선" 확인. (탑뷰 감각 직접 훈련)
- 한타 중 1초에 한 번 Tab 힐끗.
- 진입 전 자문: "나를 보러 올 수 있는 적 몇 명? 이탈 경로 어디?"

## 목표 플레이어 2 — PROGEN (아시아 챔피언 3 겐지)
- 영상: https://www.youtube.com/watch?v=sj0L-Zjchu8&t=518s
- **운영 핵심** (내가 지향하는 방식):
  - **상대의 자리 전개를 먼저 차단한다.** 사이드를 걷는 것도 이 목적의 수단 — 상대가 자리 잡고 전개하기 *전에* 끊어 셋업을 무산시킴 (사후 대응 X, 선제 차단 O).
  - **선제적으로 사이드를 걷고 본대에서 운영.** (단 '걷어내기'지 '추격'이 아님 — 시간 통제 O)
  - **기본은 정면**에서 겐지 운영. **아군이 리그룹할 때** 사이드로 빠져 **정면 압박을 풀어줌** (아군이 다시 모일 시간을 버는 타이밍 플랭크 — 무작정 사이드 X).
  - **양각(크로스파이어)은 형성하되, 트레이서처럼 사이드를 깊게 파진 않음.** 본대와 연결이 끊기지 않는 **얕은 양각** 유지 (겐지는 트레만큼 재진입·도주가 자유롭지 않으므로).
  - **접근 시 벽타기로 상대 시야에서 벗어난 상태로 붙음** (내가 이미 쓰는 테크닉과 동일 — 이 부분은 방향이 맞다는 확인).
  - **의도적으로 탱커 어그로를 받아 상대 스킬을 빼내고**, 본인은 **튕겨내기(디플렉트)로 생존.**
  - **튕겨내기가 빠졌을 땐 탱커가 어그로를 받아주는** 식으로 쿨기를 교대 → 생존 유지.
- **관찰 포인트**:
  - 사이드에 몇 초 쓰고 언제 본대로 복귀하나 (시간 통제)
  - 디플렉트를 '생존'이 아니라 '상대 스킬 빼내는 미끼'로 쓰는 타이밍
  - 디플렉트 쿨 동안엔 어떻게 포지션/탱커 뒤에 숨나 (쿨기 교대)
- **주의(내 실전과 연결)**: 남극 반도 판의 벤처 '추격' 실패와 대비 — 사이드는 걷되 시간 통제. [[patterns]] "정보 없이 자원 먼저 던지지 마라" 원칙과 같은 결.
