# 오버워치 셀프 피드백

경기별로 플레이를 돌아보고 개선점을 누적 기록하는 공간입니다. (역할군: 딜러 / 탱커)

> ## 🧠 마인드셋
> **1. "진판에서는 하나씩만 배워가자."**
> 한 경기에 모든 걸 고치려 하지 않는다. 매 경기 배워갈 것 딱 하나에만 집중한다.
>
> **2. "남탓·정치는 쓰레기 데이터다."**
> 내 플레이 개선에 도움 안 되는 무분별한 비판, 남탓, 정치는 생각도 분석도 하지 않는다.
> 오직 **내가 통제할 수 있는 것**만 본다. 좌절될 땐 이 문장을 떠올린다.
>
> **3. "통제 가능한 것엔 최선을 다하되, 통제 불능인 것도 있다."**
> 내가 통제할 수 있는 것에 대해서는 최선을 다하되, **아무리 내가 잘하고 통제력이 높아도 통제 불능인 것들이 있다.**
> 팀운·밸런스패치처럼 못 바꾸는 건 내 잘못이 아니다. 남탓도, 자책도 하지 않는다.
> 팀운이나 게임 때문에 스트레스받을 땐 이 사실을 떠올린다.

## 사용법

1. 경기가 끝나면 [template.md](template.md) 내용을 복사합니다.
2. `logs/` 폴더에 `YYYY-MM-DD-맵이름.md` 형식으로 새 파일을 만들거나, [log.md](log.md)에 이어 붙입니다.
3. 항목을 채우고, 이번 경기의 **핵심 개선점 1가지**를 꼭 정합니다.
4. 주기적으로 [log.md](log.md)를 훑어보며 반복되는 실수 패턴을 찾습니다.

## 파일 구성

- [log.md](log.md) — 경기별 피드백 누적 로그
- [template.md](template.md) — 복사해서 쓰는 빈 템플릿
- [patterns.md](patterns.md) — 반복되는 실수/개선 포인트 모음 (주기적으로 정리)
- [references.md](references.md) — 지향하는 플레이스타일/레퍼런스 (SUEL 겐지 등)

## 기록 원칙

- **구체적으로**: "포지션 안 좋았음"보다 "2번째 한타에서 사이트라인 노출된 채로 궁 없이 진입".
- **결과보다 판단**: 이기고 졌는지보다, 그 순간 내 선택이 옳았는지에 집중.
- **1경기 1개선점**: 한 번에 다 고치려 하지 말고 다음 경기에 적용할 것 하나만.
