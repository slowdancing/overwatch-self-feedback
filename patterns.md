# 반복 패턴 모음

여러 경기의 로그를 훑으며 **반복되는 실수**와 **잘 통하는 습관**을 여기에 모읍니다.

## 자주 나오는 실수
| 패턴 | 언제 발생 | 교정 방법 |
|------|-----------|-----------|
| 이탈 경로 없이 진입 → 사망 | 네팔(질풍 개활지), 도라도(에코 어그로) | 진입 전 이탈 경로·높이 먼저 확보. 안 되면 진입 보류 |
| 스킬을 상대 시야 안에서 씀 | 질풍참·강펀치 | 상대가 나를 안 볼 때/시선 안 끌렸을 때 꽂기. 강펀치는 짧게 차징 |
| 강펀 차면 급하게 진입 (천천히 못함) | 도라도, 사모아 | 각 없으면 억지 침투 X. 탱커 걷어 '자리' 확보 먼저 → 뒷라인 각 열림 |
| 피 깎이는 것에 대한 두려움 | 교전 중 | 풀피 강박 버리기, 깎이면 질풍/도주 후 힐팩 |
