# 반복 패턴 모음

여러 경기의 로그를 훑으며 **반복되는 실수**와 **잘 통하는 습관**을 여기에 모읍니다.

## 🧭 핵심 깨달음 — 모든 실수는 한 뿌리
지금까지 판마다 다르게 보였던 실수들이 결국 **"상대 위치를 먼저 파악했는가"** 하나로 수렴한다.
- 네팔 질풍 — *나를 볼 수 있는 상대*를 알아야 시야 밖 진입
- 도라도·사모아 강펀 — *상대가 나를 보는지*를 알아야 타이밍·자리 판단
- 왕의 길 나노 — *적 힐러가 어디 있는지*를 알아야 제대로 문다

→ 스킬 실수처럼 보였지만 실은 **정보 부족**이 원인.

**실행 방식** (큰 깨달음이라도 한 번에 목표 삼지 않기 — "하나씩" 원칙)
- **진판 중**: 지금 잡은 구체적 행동 하나만 (현재: "나노 받으면 힐러부터").
- **복기 루틴(장기)**: 리플레이를 적 시점/탑뷰로 돌려 "이 순간 적들이 어디 있었나" 확인. 쌓이면 "항상 파악"이 저절로 됨. (→ references.md 맵 인지)

## 자주 나오는 실수
| 패턴 | 언제 발생 | 교정 방법 |
|------|-----------|-----------|
| 이탈 경로 없이 진입 → 사망 | 네팔(질풍 개활지), 도라도(에코 어그로) | 진입 전 이탈 경로·높이 먼저 확보. 안 되면 진입 보류 |
| 스킬을 상대 시야 안에서 씀 | 질풍참·강펀치 | 상대가 나를 안 볼 때/시선 안 끌렸을 때 꽂기. 강펀치는 짧게 차징 |
| 강펀 차면 급하게 진입 (천천히 못함) | 도라도, 사모아 | 각 없으면 억지 침투 X. 탱커 걷어 '자리' 확보 먼저 → 뒷라인 각 열림 |
| 자원(나노) 써서 동수 교환 | 왕의 길 | 나노는 팀이 창을 현금화할 때만. 타겟은 힐러 우선(아나 ≥ 힐량 높은 힐러). 단 **아나 수면총 빠졌을 때만 직접 문다** — 살아있으면 빼거나 차선 타겟 |
| 피 깎이는 것에 대한 두려움 | 교전 중 | 풀피 강박 버리기, 깎이면 질풍/도주 후 힐팩 |
| 팀 열세 → 하드캐리 강박 → 무리 진입 | 서킷 로얄(팀 열세 인지 후) | 약팀일수록 원칙 강화·더 안 죽기. 캐리=이득 전환+생존, 던지기 X. 통제 가능한 건 내 데스뿐 |
| 겐지 조급증 (에임 안 맞았는데 표창 발사) | 겐지 교전 전반(고질) | 막 쏘는 게 오히려 느림(0딜). "확인→발사" 순서 + 데스매치 훈련. 근본은 셋업 없이 붙어 패닉 → 이탈로 있을 때 붙기 |

## 집중 훈련 중인 것
- **질풍참 = 위치 잡는 기술** (맞히는 기술 아님). 긋기 전 반드시 상대 시야에서 벗어나기.

## 🎯 다음 겐지 게임 플랜 (PROGEN 기본값)
흐름(몸으로): **사이드 파는 서브딜러 있으면 걷기 → 본대에서 양각 형성 → 상대 탱커 세게 압박 → 디플렉트로 탱커 스킬 빼기(베스트) → 힐러 시선이 탱커 힐에 뺏겼을 때 벽타고 고지대 진입.**
- **가드레일**: 고지대 진입해도 **질풍참(이탈로)은 남겨둔다.** 진입 = 올인 X, 각 아니면 질풍 복귀.
- **게임 중 체크포인트(딱 하나)**: *"나 지금 본대랑 연결돼 있나?"* — 죽거나 위험할 때마다 이것만 자문.
- SUEL식 뒷라인 딥다이브는 이번엔 봉인 (위치 파악 완성되면 해금).
