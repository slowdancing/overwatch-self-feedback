# 반복 패턴 모음

여러 경기의 로그를 훑으며 **반복되는 실수**와 **잘 통하는 습관**을 여기에 모읍니다.

## 🧭 핵심 깨달음 — 모든 실수는 한 뿌리
지금까지 판마다 다르게 보였던 실수들이 결국 **"상대 위치를 먼저 파악했는가"** 하나로 수렴한다.
- 네팔 질풍 — *나를 볼 수 있는 상대*를 알아야 시야 밖 진입
- 도라도·사모아 강펀 — *상대가 나를 보는지*를 알아야 타이밍·자리 판단
- 왕의 길 나노 — *적 힐러가 어디 있는지*를 알아야 제대로 문다

→ 스킬 실수처럼 보였지만 실은 **정보 부족**이 원인.

**실행 방식** (큰 깨달음이라도 한 번에 목표 삼지 않기 — "하나씩" 원칙)
- **진판 중**: 지금 잡은 구체적 행동 하나만 (현재: "나노 받으면 힐러부터").
- **복기 루틴(장기)**: 리플레이를 적 시점/탑뷰로 돌려 "이 순간 적들이 어디 있었나" 확인. 쌓이면 "항상 파악"이 저절로 됨. (→ references.md 맵 인지)

## 📏 실력을 무엇으로 재는가 — 킬이 아니다
**"적을 잡았는가"는 탱커에게 최악의 지표.** 증거: 왕의 길 18킬 2데스 패배 / 네온 교차로 41킬 26940딜 패배. 킬은 잘한 **결과로 따라오는 부산물**이지 잘했다는 증거가 아님.

**대신 볼 것**
- **스페이스**: 나 때문에 적이 자리를 내줬나?
- **쿨기 강제**: 적 방어 자원(수면·그랩·버블·궁)을 빼냈나?
- **생존**: 안 죽고 매 한타에 존재했나? (탱커가 살아있는 것 자체가 밸류)
- **아군 활성화**: 내 덕에 우리 딜러가 각을 잡았나?
- **한타 승률 + 목표 전진**: 킬 누가 냈든 **우리가 그 한타를 이기고 밀었나?**
- 한 문장: **"내가 킬을 냈나"가 아니라 "우리가 한타를 이겼고, 내가 그걸 쉽게 만들었나".**

**결과가 아니라 판단의 질로 잰다**
- ❌ "졌으니 내가 못했다" / "20킬 냈으니 잘했다" (결과엔 통제 밖 노이즈가 섞임)
- ✅ **"그 순간 내가 가진 정보로, 그게 최선의 선택이었나?"** (→ 아래 정보 테스트)

**참고 — 실력의 정의**: 마스터가 골드에 가도 **아무것도 못 하는 판은 나온다**(딜러가 안 쏘면, 팀이 먼저 짤리면 개인기로 못 살림. 저티어일수록 랜덤해서 오히려 읽고 대응이 안 됨). 차이는 **빈도**일 뿐. → **실력 = 못 이기는 판을 0으로 만드는 게 아니라, 영향력을 행사할 수 있는 판의 비율을 올리는 것.** (마인드 3번의 증거: 통제 불능은 실력 부족이 아니라 게임의 성질)

## ⚖️ 내 통제 vs 통제 불능 — 구분법
"더 잘할 수 있었던 것"과 "내 통제 밖이었던 것"을 가르는 4가지 테스트.

1. **거슬러 올라가기 테스트 (제일 강력)** — 나쁜 결과에서 거꾸로 짚으며 "여기서 내 선택이 있었나?" 물음. **마지막으로 내 선택이 있던 지점 = 내 통제의 경계.**
   - "솜브라 해킹당해 녹음" → 해킹은 못 막지만, 한 단계 위 **"왜 해킹당하면 죽는 위치에 있었나"** = 내 통제.
   - "토르가 궁 들고 죽음" → 거슬러도 **내 선택으로 이어지는 고리 없음** = 내 통제 아님. 끝.
2. **정보 테스트** — 그 순간 가진 정보로 판단이 합리적이었나? 합리적이었으면 **결과가 나빠도 실수 아님.** (예: 아나 위치 모르고 캐서디 문 것 = 정보 부족이지 실수 X. 결과로 판단을 소급해 까지 말 것.)
3. **반복 테스트 (신뢰도 최고)** — **한 판으로는 구분 불가**(노이즈). 팀·맵·조합이 다 바뀌었는데도 계속 나타나면 **상수는 나 = 내 통제.** 한 번뿐이면 상황/변수. → **이 파일이 곧 구분 장치.**
4. **구체적 대안 테스트** — "그 순간 했어야 할 다른 행동"을 구체적 한 문장으로 말할 수 있나?
   - ✅ "벽 끼고 파워블락 썼어야" = 내 통제. ❌ "그 한타를 이겼어야" = 통제 밖이거나 아직 못 찾음.

**기본 자세**: 일단 통제 가능한 부분을 찾아본다(내 탓이라서가 아니라 그것만이 도움되니까). 근데 정직하게 찾아도 **없으면 없는 것** — 멈추고 넘긴다. 억지 자책거리 만들지 말 것. (마인드 3번)

## 🚦 안전 vs 공격 — 언제 무엇을
둘은 반대가 아니라 **순서**. **안전으로 조건을 만들고 → 공격으로 현금화.** (안전 = 소극이 아니라 공격할 창을 버는 작업: 살고·자리 잡고·상대 쿨기 빼고·궁 모으기)
- ⚠️ 공격적 ≠ 자원 많음. 던지면 자원 마이너스(적 궁만 채움). "안전 vs 공격"은 자원 총량이 아니라 **타이밍** 문제.
- **공격 초록불 체크** (켜지면 공격, 아니면 안전=초록불 만들기):
  1. 이점 있나? 픽/인원 우위(5:4) OR **적 핵심 쿨기 빠짐**(수면·그랩·버블)
  2. 내 도구 있나? 강펀/궁/이탈기
  3. 팀이 따라오나? (혼자 X)
  4. 실패해도 살 각(이탈로) 있나?
- **여러 개 켜질수록 확실한 공격. 하나도 없으면 그건 공격이 아니라 도박.**
- **나의 기본값: 디폴트 안전, 공격은 초록불로 해금** (내 데스는 "초록불 없는데 공격"에서 나옴).

## 자주 나오는 실수
| 패턴 | 언제 발생 | 교정 방법 |
|------|-----------|-----------|
| 이탈 경로 없이 진입 → 사망 | 네팔(질풍 개활지), 도라도(에코 어그로) | 진입 전 이탈 경로·높이 먼저 확보. 안 되면 진입 보류 |
| 스킬을 상대 시야 안에서 씀 | 질풍참·강펀치 | 상대가 나를 안 볼 때/시선 안 끌렸을 때 꽂기. 강펀치는 짧게 차징 |
| 강펀 차면 급하게 진입 (천천히 못함) | 도라도, 사모아 | 각 없으면 억지 침투 X. 탱커 걷어 '자리' 확보 먼저 → 뒷라인 각 열림 |
| 자원(나노) 써서 동수 교환 | 왕의 길 | 나노는 팀이 창을 현금화할 때만. 타겟은 힐러 우선(아나 ≥ 힐량 높은 힐러). 단 **아나 수면총 빠졌을 때만 직접 문다** — 살아있으면 빼거나 차선 타겟 |
| 피 깎이는 것에 대한 두려움 | 교전 중 | 풀피 강박 버리기, 깎이면 질풍/도주 후 힐팩 |
| 팀 열세 → 하드캐리 강박 → 무리 진입 | 서킷 로얄(팀 열세 인지 후) | 약팀일수록 원칙 강화·더 안 죽기. 캐리=이득 전환+생존, 던지기 X. 통제 가능한 건 내 데스뿐 |
| 겐지 조급증 (에임 안 맞았는데 표창 발사) | 겐지 교전 전반(고질) | 막 쏘는 게 오히려 느림(0딜). "확인→발사" 순서 + 데스매치 훈련. 근본은 셋업 없이 붙어 패닉 → 이탈로 있을 때 붙기 |

## 🎯 다음 겐지 게임 플랜 (PROGEN 기본값)
흐름(몸으로): **사이드 파는 서브딜러 있으면 걷기 → 본대에서 양각 형성 → 상대 탱커 세게 압박 → 디플렉트로 탱커 스킬 빼기(베스트) → 힐러 시선이 탱커 힐에 뺏겼을 때 벽타고 고지대 진입.**
- **가드레일**: 고지대 진입해도 **질풍참(이탈로)은 남겨둔다.** 진입 = 올인 X, 각 아니면 질풍 복귀.
- **게임 중 체크포인트(딱 하나)**: *"나 지금 본대랑 연결돼 있나?"* — 죽거나 위험할 때마다 이것만 자문.
